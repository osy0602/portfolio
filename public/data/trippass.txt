트립패스는 AI를 활용하여 최적의 여행 일정을 추천하는 플랫폼입니다. 
사용자의 선호도를 분석하여 맞춤형 여행 계획을 생성합니다.

🔧 주요 기능:
✔ AI 기반 맞춤 여행 일정 추천
✔ 실시간 여행 크루 매칭
✔ Google Map 지도 연동

🚨 문제점 & 해결 방법:
기존 여행 앱들은 일정 추천 기능이 부족하여 사용자가 직접 모든 계획을 세워야 하는 불편함이 존재
여행 장소, 날짜만 입력한다면 개인의 여행 성향을 반영하여 여행지와 식당, 카페 등 다양한 장소를 OpenAI가 추천!

🌟 차별화 포인트:
AI를 활용한 맞춤형 일정 추천 가능
해당 여행 장소에 방문하는 다른 여행자들을 찾아 여행 크루 매칭 기능 이용 가능

⚙ 사용 기술:
- Python, Javascript
- FastAPI, React
- MySQL, MongoDB
- OpenAI, Gemini, SerpAPI, openweathermap
- Docker, Github
- AWS EC2

📅 프로젝트 기간: 2024.6 ~ 2024.8
