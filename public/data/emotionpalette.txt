Emotion Palette는 사용자의 뇌파 데이터를 분석하여 감정을 분류하고, 이를 바탕으로 개인화된 미디어 아트를 제공하는 인터랙티브 프로젝트입니다.

🧠 감정을 데이터화하고 시각적으로 표현하는 새로운 방식의 체험형 미디어 아트입니다.

🔧 주요 기능:
✔ 뇌파 데이터를 기반으로 감정 분류 (딥러닝 활용)
✔ 감정 데이터 시각화 (그래프 및 감정 컬러칩 제공)
✔ 개인 맞춤형 미디어 아트 생성
✔ 핸드 트래킹을 통한 동작 기반 인터랙션 기능
✔ 감정 데이터를 기반으로 한 미디어 아트 시뮬레이션

🚨 문제점 & 해결 방법:

❌ 문제점
기존 감정 분석 기술은 텍스트, 음성, 표정을 분석하는 방식이 대부분으로, 보다 직관적이고 몰입감 있는 방식이 부족함.
감정 데이터를 단순한 수치로만 제공하는 것이 아니라, 사용자가 즉각적으로 체감할 수 있는 시각적 피드백이 필요함.

✅ 해결 방법
뇌파 데이터를 활용하여 감정을 직접 측정하고, 이를 시각적으로 변환하여 즉각적인 피드백 제공.
감정 분석 결과를 단순한 데이터가 아니라, 미디어 아트(색상, 형태, 애니메이션)로 표현하여 사용자 경험 극대화.
핸드 트래킹 기술을 활용하여 사용자가 자신의 감정을 직접 상호작용하며 체험할 수 있도록 설계.


🌟 차별화 포인트:

✅ 뇌파 데이터를 활용한 감정 분석
일반적인 감정 분석(AI 텍스트 분석, 표정 인식)과 달리, 사용자의 실제 신경 신호(EEG)를 활용하여 감정을 분석.
감정 데이터는 딥러닝 모델을 통해 분류되며, 실시간으로 시각화됨.

✅ 개인 맞춤형 미디어 아트 제공
감정 분석 결과에 따라 각 사용자만의 감성 컬러칩을 생성하여 맞춤형 아트워크를 구현.
감정을 반영한 실시간 애니메이션 및 인터랙티브 UI 제공.

✅ 핸드 트래킹을 활용한 몰입형 체험
감정 컬러 데이터를 기반으로 사용자가 손을 움직이면 반응하는 미디어 아트 구현.
미디어 아트를 조작할 수 있는 직관적인 UX 제공.


⚙ 사용 기술:
Python, C# (딥러닝 모델 및 데이터 분석)
MediaPipe, Unity (핸드 트래킹 및 미디어 아트 구현)
Firebase (실시간 데이터 저장)
UDP, TCP, Socket 통신 (디바이스 간 데이터 전송)
